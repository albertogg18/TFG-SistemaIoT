# SmartPlant 🌱

Sistema IoT para la monitorización y riego automático de plantas domésticas.

## 📌 Características
* 📡 **Telemetría:** Humedad, temperatura y luz en tiempo real.
* 💧 **Modos de Riego:** Manual, por reglas (umbrales) o Inteligencia Artificial (Google Gemini).
* 🛡️ **Seguridad:** Sistema anti-ahogos integrado para proteger la planta.
* 💻 **Web:** Dashboard interactivo con historial y gráficas.

## 🛠️ Tecnologías
* **Frontend:** React, TypeScript, Tailwind CSS.
* **Backend:** Node.js, MongoDB.
* **Hardware:** ESP32, Relé 5V, Bomba de agua.
